<h2>Excited to share my latest project! 📚<h2>

<p>I have successfully designed and developed a fully functional Library Management System. This project was aimed at digitizing core library operations to make them more efficient and user-friendly.

Key Modules:

Admin Panel: Complete control to manage the system.

User/Librarian Module: For handling day-to-day tasks.<p>

<h3>Core Functionalities:<h3>

Book Management: Add, update, delete, and search for books by title, author, or genre.

Member Management: Registration and management of library members.

Issue/Return Processing: Streamlined workflow for issuing books and calculating fines for late returns.
